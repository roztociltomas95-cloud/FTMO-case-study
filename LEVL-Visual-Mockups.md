## 5. TECHNICAL SPECIFICATIONS SUMMARY

### Logo Files Needed

**SVG (Vector):**
- `LEVL_Logo_Horizontal_Color.svg` - Icon + wordmark, gradient
- `LEVL_Logo_Horizontal_White.svg` - Icon + wordmark, white
- `LEVL_Logo_Stacked_Color.svg` - Icon above wordmark
- `LEVL_Icon_Color.svg` - L icon only, gradient
- `LEVL_Icon_White.svg` - L icon only, white
- `LEVL_Wordmark_Color.svg` - "LEVL" text only

**PNG (Raster):**
- `LEVL_AppIcon_1024x1024.png` - iOS app icon
- `LEVL_AppIcon_512x512.png` - Android app icon
- `LEVL_Logo_Horizontal_@1x.png` (baseline)
- `LEVL_Logo_Horizontal_@2x.png` (retina)
- `LEVL_Logo_Horizontal_@3x.png` (high-res)

**Favicon:**
- `favicon.ico` - 16x16, 32x32, 48x48 (multiple sizes in one file)
- `favicon-32x32.png`
- `favicon-16x16.png`

### Minimum Sizes

**Horizontal Logo:**
- Digital: 120px width minimum
- Print: 1 inch width minimum

**Icon Only:**
- Digital: 80px minimum
- Print: 0.5 inch minimum

**Clear Space:**
- 0.5x logo height on all sides

---

## USAGE NOTES

**DO:**
✅ Use gradient on dark backgrounds (primary)
✅ Use solid green or white on light backgrounds
✅ Maintain aspect ratio (don't stretch)
✅ Ensure clear space around logo
✅ Use SVG when possible (scalability)

**DON'T:**
❌ Rotate or distort logo
❌ Change gradient colors
❌ Add effects (shadows, outlines) unless specified
❌ Place on busy backgrounds (reduce logo visibility)
❌ Use red or other off-brand colors
